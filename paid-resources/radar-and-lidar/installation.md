# Installation

## Installing The Item

***

* Head over to ox\_inventory/data/weapons.lua and paste the following item

```lua
['WEAPON_PROLASER4'] = {
	label = 'Lidar',
	weight = 700,
	durability = 0.1,
},
```

* Locate the image for the item in brazzers\_radar/images and drag and drop that into ox\_inventory/web/images
