# Exports

### blockFreeCam

***

Enable or disable the usage of freecam.

```lua
exports.brazzers_perspective:blockFreeCam(block)
```

* block: `boolean`
  * true
  * false

### blockZoom

***

Enable or disable the usage of zoom.

```lua
exports.brazzers_perspective:blockZoom(block)
```

* block: `boolean`
  * true
  * false
